# ClearCut вҖ” AI Background Removal

**ClearCut** is a premium, 100% client-side background removal tool. Powered by AI and optimized for speed and privacy, it allows users to remove image backgrounds directly in the browser without ever uploading data to a server.

**ClearCut** бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһӣбһ»бһ”бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ AI бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бҹЎбҹ бҹ % бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһ—бһ¶бһ–бһҡбһ бҹҗбһҹ бһ“бһ·бһ„бһҜбһҖбһҮбһ“бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ бһҠбҹ„бһҷбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҡбһјбһ”бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһЎбһҫбһҷбҹ”

- **High-End UI**: Responsive, glassmorphism-inspired design reflecting modern aesthetics. (бһҡбһ…бһ“бһ¶бһ”бһҗбһ‘бҹҶбһ“бһҫбһ”бҹ– бһҹбҹ’бһҡбһҹбҹӢбһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһӣбһҫбһӮбҹ’бһҡбһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ)
- **Khmer Support**: Fully localized in Khmer with the beautiful Kantumruy Pro font. (бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһҒбҹ’бһҳбҹӮбһҡбҹ– бһ”бһ¶бһ“бһ”бһҖбһ”бҹ’бһҡбҹӮбһ–бҹҒбһүбһӣбҹҒбһү бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ бһҖбһ“бҹ’бһ‘бһ»бһҷбһ бҹ’бһҳ Pro)
- **Instant Downloads**: Save your results as high-quality PNGs with automatic timestamps. (бһ‘бһ¶бһүбһҷбһҖбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ– бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶ PNG бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ)

## рҹҡҖ Tech Stack | бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶

## рҹӣ пёҸ Getting Started | бһҡбһ”бҹҖбһ”бһҠбҹҶбһЎбһҫбһ„

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Lachsokhour/clearcut.git
   cd clearcut
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## рҹ”’ Security & Privacy | бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ бһ“бһ·бһ„бһҜбһҖбһҮбһ“бһ—бһ¶бһ–

This application leverages **ONNX Runtime Web** to run machine learning models directly in your browser's WASM environment. Your images are converted to blobs locally, processed by the AI, and the result is generated as a local URL. No tracking, no storage, no server-side logic.

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **ONNX Runtime Web** бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ AI бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҡбһјбһ”бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһЎбһҫбһҷбҹ” бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбҹ”
